# Research Paper Package

This folder contains a starter publication bundle for the project.

## Files

- `manuscript.tex`: LaTeX manuscript template for journal or conference submission.
- `references.bib`: BibTeX file for citations.
- `ethics-and-data-availability.md`: Notes for ethics, consent, and data handling.
- `research-proposals.md`: Future-work and study proposal ideas for the paper.
- `submission-checklist.md`: Pre-submission checklist.
- `figures/README.md`: Placeholder for exported figures and tables.

## Suggested Workflow

1. Fill in the manuscript sections with final experiment details.
2. Replace any placeholder text in the bibliography with verified citations.
3. Export charts and screenshots into `paper/figures/`.
4. Run your final proofreading and formatting pass before submission.
